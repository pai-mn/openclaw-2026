## Import Rules

- Use `@/*` path aliases for all imports (e.g., `@/components/...`, `@/i18n/...`)
- The `@` alias should point to the `src` directory

## Tech Stack

- Bun
- Next.js 16
- React 19
- Tailwind CSS 4
- Lucide React (icons)
- js-yaml (for i18n translations)
